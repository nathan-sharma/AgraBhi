Information

The proram will refuse to log if your sample points are too close together. This is to account for GPS noise. To bypass this, use Force Log.

Please see Mission Planner for telemetry data. The data hub is only for data collection and actuator control.

Enter the coordinates of your sprinklers to improve the accuracy of the heatmmap.

Before logging a point, we recommend clicking 'Read Sensors' to make sure you're still getting data.
